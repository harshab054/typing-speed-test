A fun and interactive **Typing Speed Test** built with **HTML, CSS, and JavaScript**. Test your typing speed and accuracy with randomly generated paragraphs.

## Features
- Dynamic paragraph display
- Highlighting correct, incorrect, and current characters
- Timer starts automatically
- Calculates WPM (Words Per Minute)
- Calculates Accuracy (%)
- Restart button with random paragraph
- Cool animations and glow effects

## How to Use
1. Open `index.html` in a browser.
2. Start typing in the input box.
3. Watch the timer, WPM, and accuracy update in real-time.
4. Click **Restart** to try a new paragraph.
